# Brasileirao

![](https://img.shields.io/github/size/RaquelCCabral/Brasileirao/index.html?style=flat)

<label> ⚽ Descrição:

<p> Página usada para verificar a tabela do Campeonato Brasileiro em tempo real, com uso de API do site: https://www.api-futebol.com.br/

<label> ⚽ Instalação e Execução:

<li> Para instalar em seu computador você precisa clicar no botão verde escrito 'Code' e em seguida selecionar a opção 'Download Zip'. Ou, se preferir:
<p align = 'center'><a href="https://github.com/RaquelCCabral/Brasileirao/archive/refs/heads/main.zip" target="_blank"><img src="https://img.shields.io/badge/-Download-%230077B5?style=for-the-badgelogoColor=white" target="_blank"></a>
<li> Para executar é necessário que você tenha alguma ferramenta de extração de arquivos, como a winRAR, para extrair a pasta para o local no seu computador
que você deseja ter acesso. Depois, basta clicar no arquivo 'index.html'.
